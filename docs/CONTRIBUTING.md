# Contributing to Ribossome

Thanks for your interest in contributing! Ribossome is an ambitious GPU ALife project, and help is appreciated.

## How to Contribute

1. **Fork** the repository.
2. Create a branch: `git checkout -b my-feature`.
3. Make your changes (follow Rust style: `cargo fmt` and `cargo clippy`).
4. Test thoroughly (run in release mode: `cargo run --release`).
5. Submit a **Pull Request** with a clear description.

## Guidelines

- Keep the single-shader philosophy where possible (but refactor for clarity/performance).
- Respect the MIT license.
- Be kind and constructive in issues/PRs.

## Reporting Bugs / Suggestions

Open an issue with details (OS, GPU, steps to reproduce, screenshots/videos).
